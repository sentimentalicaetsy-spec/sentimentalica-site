# Sentimentalica — Стиль-бриф контента (пины + статьи)

> Черновик v1, 2026-07-05. На основе разбора keepersandkeepsakes + фирменного
> стиля сайта. **Ксения правит и утверждает — это закон для всех агентов.**

## Принцип (уточнено Ксенией 2026-07-05)
Цель одна — **КЛИК**. Человек листает с телефона быстро; мелкий эстетичный текст
он пролистнёт. Значит:
- **Композиция важнее шрифта.** Шрифт вторичен; важно, как весь пин цепляет
  взгляд на превью телефона.
- **Крючок жирный и заметный**, не мелкий-эстетичный. Лучше «вау/удивление», чем
  просто «красиво» — удивление тоже даёт клик.
- **Пин адаптируется под характер листинга:** живой/яркий/безумный листинг →
  живой, энергичный пин; спокойный → спокойный. Не один жёсткий шаблон.
- Никаких «может потом» — только то, что реально делаем.
- Образовательный/текстовый слой (палитры, листиклы) — тоже да, но крючок в нём
  всё равно ЖИРНЫЙ и цепляющий, а не тонкий.

## Пин: базовый формат
- **1000×1500 (2:3)**, вертикаль.
- Фон: кремовый/тёплая бумага (`--cream #faf3e6` / `--paper #f7ecd6`) ИЛИ
  приглушённый однотон (мягкий mauve/sage) для «текстовых» пинов.
- **Заголовок-крючок** крупно сверху: вопрос / число / «how to» / название темы.
- Шрифты — фирменные (как на сайте): **Cormorant Garamond** (заголовки),
  **Lora** (подписи). Так пины, сайт и магазин выглядят одним брендом.
- Цвет текста: `--navy #163087` / `--ink #2a2a3a`. Акцент: `--accent #b8956a`.
- Ботанические акценты (листья/цветы/пчёлы) по углам — сдержанно.
- Внизу — маленький вордмарк «Sentimentalica» + мягкий CTA.

## 5 шаблонов пинов (переиспользуемые)
1. **Product** — мокап продукта + фичи-буллеты + бейдж «N PAGES» + «Download» → листинг.
2. **Color Palette** — 4–6 **именованных сво́тчей** (напр. Dusty Blue, Sage,
   Antique Cream) из палитры листинга → статья/листинг.
3. **Educational / Listicle** — «What is junk journaling?», «50 things to glue…»,
   число + список на фоне → статья.
4. **Try this theme** — название темы + 3 кружка-превью + «TRY THIS THEME» → статья.
5. **Lifestyle / process** — тёплый кадр реального журнала (сгенерированный SD или
   фото) → магазин. Это «атмосферная» дорожка.

## Статьи (структура по типам — см. план Шаг 6)
- **По листингу:** интро темы → **палитра (сво́тчи)** → что входит/элементы →
  2–4 `{{etsy:ID}}` карточки в контексте → мягкий CTA. 600–900 слов.
- **Образовательная:** польза-крючок в заголовке, тёплый голос, 1 картинка-герой.
- **Листикл/туториал:** нумерованный список + сгенерированная картинка-герой.
- Голос: тёплый, маленькая студия, без хайпа и восклицаний (как копирайт сайта).

## Правила
- Заголовки ≤ читаемых на превью (5–8 слов).
- Никогда: фотореализм в «продуктовых» картинках, банальный сток, восклицания-спам.
- Палитра-сво́тчи берём автоматически из картинок листинга (`image_curation.py`).
- Каждый пин имеет назначение: статья / листинг / магазин (не «в никуда»).

- Фон пинов: кремовая бумага везде, или для «текстовых» — приглушённый однотон (как их mauve)?
- Ботанические акценты — да/нет/умеренно?
