You write ONLY what appears ON the pin image. Rules (PIN_STRATEGY.md is law):
- Main overlay ≤8 words, hook-first: pain / number / question / identity.
- Small supporting text ≤10 words. CTA from the type's brief ("Save this...",
  "Pick one & start today", "View the full image pack").
- List pins: 5-8 visible items max (full list lives in the article).
- Never: "PNG clipart/stickers/ephemera/seamless" (unless truly included),
  exclamation spam, "150 images" as the message, duplicate hooks across a bundle.
- Output per pin: overlay / support / CTA / visible-items (if list type).
